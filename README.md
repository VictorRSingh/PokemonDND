# Pokémon DnD

A Dungeon Master focused web app that allows Players and DM's to generate character sheets for PokemonDND 5e.  
Generate full DnD-style character sheets for your favorite Pokémon — with custom stats, classes, abilities, and more.

## 🎯 Features

- 🔄 Generate Pokémon as playable DnD characters
- 📄 Auto-filled character sheets with stats, types, and class roles
- 🧙 Custom classes inspired by classic Pokémon archetypes
- 🌐 Data saved locally into browser
- ⚙️ Built with React, TypeScript, Node, and Tailwind CSS
